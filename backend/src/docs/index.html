CodeOrb API Documentation

🔁 POST /run

Run a code snippet in an isolated container and return the output.

Request Body (JSON):

{
  "language": "python",
  "code": "print(2 + 2)"
}

Response:

{
  "output": "4\n",
  "error": null
}

📥 Supported Languages

  • python
  • node
  • c
  • cpp
  • java

⏱️ Timeout

The server waits up to 10 seconds for a response. For longer tasks, future support for async mode may be added.

🛡️ Security & Isolation

  • Each code snippet runs in a Docker container
  • Limited CPU and memory
  • No internet/network access
  • Read-only mount of code
